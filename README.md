# Sistema de Chamados TI

Projeto desenvolvido em Python para gerenciamento de chamados de suporte técnico.

## Funcionalidades

- Cadastro de chamados
- Alteração de status
- Exclusão de chamados
- Dashboard
- Data de abertura
- Prioridade
- Técnico responsável
- Persistência de dados com JSON

## Tecnologias Utilizadas

- Python 3
- JSON
- Datetime

## Como executar

Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

Entre na pasta:

```bash
cd sistema-chamados-python
```

Execute:

```bash
python chamados.py
```
